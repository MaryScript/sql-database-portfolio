# Portfólio de Banco de Dados (SQL)

Repositório dedicado aos estudos de modelagem de dados e consultas SQL, desenvolvidos durante a graduação em **Ciência da Computação (UnDF)**.

## Projetos

### 1. Sistema de Biblioteca Universitária (`/01-biblioteca-universitaria`)
Modelagem de um sistema de gestão de acervo e empréstimos.
* **Ferramenta:** MySQL Workbench.
* **Conceitos Aplicados:**
    * **DDL:** Criação de tabelas com Chaves Primárias (PK) e Estrangeiras (FK).
    * **Integridade:** Uso de Constraints (`ON DELETE RESTRICT`) para manter a consistência.
    * **DQL:** Queries analíticas utilizando `JOIN` para relatórios e `GROUP BY` para estatísticas.
